## Tools

### `market-pulse.py` — All-in-One Crypto CLI
Zero-dependency Python. Just Python 3.8+.
```bash
python3 market-pulse.py          # Full overview
python3 market-pulse.py --json   # JSON output (pipeable)
python3 market-pulse.py --html   # Generate HTML dashboard
python3 market-pulse.py --watch  # Live refresh every 60s
python3 market-pulse.py --regime # Just regime signal
python3 market-pulse.py --polymarket  # Include prediction markets
```

### `market-pulse-dashboard.html` — Static Dashboard
Zero dependency HTML dashboard. Open in browser. Auto-refreshes.

### `self-hosted-server.py` — REST API Server
Serves market data via HTTP. Requires Python.
